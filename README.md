## pandoc-reveal.js

use pandoc to transfer markdown to reveal.js presentation slides

## reference 

[reveal.js-with-pandoc](https://github.com/yanping/reveal.js-with-pandoc.git)

since it doesn't support the 2014 reveal.js from [reveal.js](https://github.com/hakimel/reveal.js)

## get source and build step

	git clone https://github.com/ted-juan/pandoc-reveal.js.git
	cd pandoc-reveal.js
	git clone https://github.com/hakimel/reveal.js
	build.sh
	firefox slides.html
